# Reform Hub v1.1 修正版

現場写真に手描きし、案件を端末保存・画像保存・共有できるWebアプリです。

## 今回の修正

- `index.html` とロゴファイルの取り違えを防ぐため、フォルダ構成を単純化
- 必要なファイルをすべて同じ階層に配置
- GitHubへ上書きアップロードしやすい構成

## GitHubへ上書きするファイル

- index.html
- styles.css
- app.js
- manifest.json
- sw.js
- vercel.json
- icon-192.png
- icon-512.png
- README.md

ZIPそのものではなく、解凍後の上記ファイルをアップロードしてください。
