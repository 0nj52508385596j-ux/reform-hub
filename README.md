# CAINZ365 FIELD MODE v1.1

## 今回追加した機能
- 「過去の現場」ボタン
- 現場の簡易保存
- 物件名・お客様名・メモ・提案内容で検索
- 過去の写真枚数・写真タグ・提案候補を表示
- 保存した共有文をすぐ開いてコピー
- 履歴の削除
- 「このまま終了」で現場履歴へ自動保存

## 保存される内容
- 日付
- 物件名
- お客様名
- 担当者
- 写真枚数
- 写真タグ
- 提案候補
- 現場メモ
- 最後に作った共有文

## 注意
この版は端末内保存です。同じiPad・スマホ・PCなら過去履歴を見られます。
20人で同じ履歴を共有するには、次の段階でクラウド保存を接続します。

## GitHubへの入れ方
ZIPを解凍し、GitHubの reform-hub へ中の6ファイルを全部上書きアップロードしてください。
