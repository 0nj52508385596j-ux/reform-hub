# CAINZ365 FIELD MODE v2.0 Lite

既存の機能・保存ロジックを維持し、HTML/CSS中心で見た目を更新した版です。

## 変更点
- 白基調の業務アプリUI
- 赤アクセントの統一
- カード、入力欄、ボタン、写真、AI提案、履歴の視認性改善
- スマホ・iPad向け余白とボタンサイズの調整
- PWAキャッシュを v2.0 Lite 用に更新

## 変更していないもの
- 写真撮影・保存
- 音声入力補助
- AI提案判定
- 履歴・LocalStorage
- 共有・PDF・手書き

## Vercelへの反映
ZIPを展開し、現在のVercelプロジェクトのファイル一式へ上書きしてください。
古い画面が残る場合は、ブラウザを再読み込みするか、ホーム画面のアプリを一度閉じて開き直してください。
