# Reform Hub v5.2 — Five Photo Day

## 新機能
- 1物件・1日につき最大5枚
- 各写真へ個別に手書き
- 各写真へ40文字までの短い題名
- サムネイルで写真を切り替え
- 写真の追加・削除
- 共有時は最大5枚をまとめて添付
- 同じ物件名・同じ日付の記録は重複を確認し、上書き可能

## 共有経路
- お客様：LINE
- 協力業者：ANDPADチャット
- 社内：Teams
- 自分の控え：Teams

## 互換性
旧版で保存した1枚の写真は、自動的に「写真1」として読み込みます。

## 注意
端末や共有先アプリが複数画像の一括共有に対応していない場合、文章のみになることがあります。
その場合は写真を端末へ保存し、指定アプリへ追加してください。
