# CAINZ365 FIELD MODE v10 β

## 今回入れた機能
- 最初から大きい文字
- 「標準・大・特大」の文字サイズ切替
- 写真へのフリーハンド描画
- 赤・青・黄、細線・太線、元に戻す
- 写真タグ：床、外壁、屋根、雨樋、基礎、玄関、窓、水まわり等
- 写真ごとの送付指示コメント
- 共有前に送る写真をチェックして選択
- 本文に「床」等があるのに該当タグ写真がない場合の警告
- 選択写真を共有機能へ添付
- 添付共有非対応端末では、写真番号入り文章をコピー
- 「今日やること」モードは入れていません

## GitHubへの入れ方
1. ZIPを解凍
2. reform-hub を開く
3. Add file → Upload files
4. 中の6ファイルをすべてアップロード
5. Commit changes
6. Vercel反映後に開き直す

古い表示が残る場合は、ホーム画面の旧アイコンを削除し、Safariでサイトを開き直してください。
