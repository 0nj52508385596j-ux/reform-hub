4. 写真への手書き、タグ、共有
5. 老眼向けの大きな文字と特大表示

## GitHubへの入れ方
1. ZIPを解凍
2. GitHubの reform-hub を開く
3. Add file → Upload files
4. 中の6ファイルを全部アップロード
5. Commit changes
6. Vercelの反映後に再度開く
