## 主な機能
- iPadのキーボード音声入力
- 現場情報・メモの自動保存
- 写真最大20枚と写真別コメント
- 要点整理／お客様向け／業者向け／社内日報
- コピー、共有、PDF保存
- ホーム画面追加対応

## GitHubへのアップロード
1. ZIPを解凍
2. GitHubの reform-hub を開く
3. Add file → Upload files
4. 解凍した6ファイルを全部選ぶ
5. Commit changes
6. Vercel反映後にページを再読み込み

古い表示が残る場合は、ホーム画面の古いアイコンを削除し、Safariから開き直してください。
