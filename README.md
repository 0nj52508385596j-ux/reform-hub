# Reform Hub v5.1 — Role Share

## 共有先を役割ごとに固定
- お客様 → LINE
- 協力業者 → ANDPADチャット
- 社内 → Microsoft Teams
- 自分の控え → Microsoft Teams

登録画面から送信方法・メールアドレス・電話番号の選択を外し、名前だけ登録する設計に変更しました。

## 操作
共有画面では「お客様」「協力業者」「社内」を選ぶだけです。
写真がある打合せは、対応端末では写真を自動添付して端末の共有画面を開きます。

## ブラウザ上の制約
PWAからLINE、ANDPADチャット、Teamsを自動的に選択して投稿することはできません。
ボタンを押した後、端末の共有画面に表示される指定アプリを1回選びます。
ANDPADが共有先候補に表示されない端末では、文章をコピーしてANDPADチャットへ貼り付けます。
