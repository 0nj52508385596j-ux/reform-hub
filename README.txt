白背景 × 白文字 修正版

【修正箇所】
・「話して入力」ボタンの文字を濃いグレーへ変更
・「物件名・日付などを確認する」の文字を濃いグレーへ変更
・詳細欄を開いた後のラベル、入力文字、入力例も読みやすく調整
・それ以外のデザインや機能は変更しません

【適用方法】
1. ZIPを解凍
2. styles.patch.css を開く
3. 中身をすべてコピー
4. 現在の styles.css の一番最後へ貼り付け
5. 保存してGitHubへアップロード
6. スマホのブラウザを再読み込み

【まだ変わらない場合】
PWAのキャッシュが残っている可能性があります。
Chromeでページを閉じて開き直すか、ホーム画面版を一度終了して再起動してください。
